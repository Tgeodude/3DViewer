3DViewer_v1.0

    Сборка:

    • make install - установка приложения 3DViewer_v1.0
    • make unistall - удаление приложения 3DViewer_v1.0
    • make clean - очистка всех файлов сборки
    • make dist - создание архива
    • make tests - проверка модулей парсинга и аффинных преобразований с помощью тестов
    • make gcov_report - отображение покрытия кода unit тестами
    • make dvi - вывод информации о проекте

    Программа позволяет:

    • Работать с файлами типа *.obj
    • Поворачивать модель на заданный угол относительно осей X, Y, Z
    • Перемещать модель на заданное расстояние относительно осей X, Y, Z
    • Масштабировать модель на заданное значение
